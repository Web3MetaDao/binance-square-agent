# 策略信号排版规范 v2.0

## 核心原则
- 🟢 = 涨（正数、上升、多头、买入）
- 🔴 = 跌（负数、下降、空头、卖出、下跌放量）
- ⚪ = 横盘 / 中性
- 所有价格用 `$X.XXXX` 格式
- 百分比统一带 `±` 号

## 六、情绪符号对照表

| 符号 | 含义 | 使用场景 |
|------|------|---------|
| 🔥 | 极度活跃/放量爆拉 | 量比>10x 或 涨>20% |
| 💥 | 崩盘/暴跌 | 跌>15% |
| 🚀 | 强力反弹 | V反>20% |
| 📈 | 稳步上涨 | 涨5-15% |
| 📉 | 稳步下跌 | 跌5-15% |
| 💤 | 横盘/缩量 | 波幅<3% |
| ⏳ | 等待确认 | 信号待验证 |
| ⚠️ | 风险警告 | 超买/死叉/恐慌 |
| 🧊 | 冷静期 | 暴跌后修复 |
| 🎯 | 关键价位 | 阻力/支撑 |
| 💰 | 价格信息 | 价格/成交额 |
| 📊 | 数据/量价 | 成交量/波幅 |
| 🔄 | 趋势反转 | 金叉/死叉 |
| 🏆 | 排名/第一 | TOP1 |
| 🫸🫷 | 信号卡片装饰 | 左右包裹币名 |

## 七、排版规则

1. 所有数据行用 `符号 内容` 空格对齐
2. 百分比统一前缀：🟢+数 / 🔴-数
3. 价格统一 `$X.XXXX` 四位小数（>1的币用两位）
4. 分隔线用 `━`（长）或 `─`（短）
5. 一组数据用 `━━━━━` 包裹，视觉分组
6. 涨跌幅排行用 `1️⃣2️⃣3️⃣` 序号
7. 关键数字加粗 `**数字**`
8. K线图用竖线 `｜` 连接，时间段用 `──` 分隔
